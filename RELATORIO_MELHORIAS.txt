✓ MELHORIA #1: Classe BaseDadosLivros (NOVA!)
────────────────────────────────────────────────
Funcionalidade: Simula um banco de dados com informações de livros

Componentes:
• Classe estática com HashMap de ISBNs
• 6 livros clássicos pré-cadastrados:
  - 978-0451524935: "1984" - George Orwell
  - 978-0062316097: "O Hobbit" - J.R.R. Tolkien
  - 978-8532511010: "Dom Casmurro" - Machado de Assis
  - 978-8535930689: "O Cortiço" - Aluísio Azevedo
  - 978-0451526342: "Orgulho e Preconceito" - Jane Austen
  - 978-8501045534: "Memórias Póstumas de Brás Cubas" - Machado de Assis

Métodos principais:
  public static LivroInfo buscarPorIsbn(String isbn)
    → Busca um livro pelo ISBN, retorna LivroInfo ou null
  
  public static boolean isbnExiste(String isbn)
    → Verifica se um ISBN existe na base de dados
  
  public static void listarTodosIsbn()
    → Exibe todos os ISBNs disponíveis com informações

Como funciona:
1. Usuario digita o ISBN (exemplo: 978-0451524935)
2. Sistema busca automaticamente na BaseDadosLivros
3. Se encontrado, exibe as informações (título, autor, editora, ano)
4. Usuário confirma e o livro é adicionado à biblioteca
5. Todas as informações são preenchidas automaticamente!

Benefício: Rápido, seguro e sem erros de digitação

✓ MELHORIA #2: Classe Usuario (IMPLEMENTADA)
──────────────────────────────────────────────
Atributos:
  • String nome: Nome do usuário
  • String email: Email para contato
  • String telefone: Telefone (opcional)
  • ArrayList<Livro> livrosEmprestados: Controle de empréstimos

Métodos implementados:

  1. emprestarLivro(Livro livro)
     - Verifica se o livro está disponível
     - Adiciona à lista do usuário
     - Marca livro como indisponível

  2. devolverLivro(Livro livro)
     - Remove livro da lista
     - Marca livro como disponível
     - Valida se o usuário realmente tem o livro

  3. listarEmprestimos()
     - Mostra todos os livros emprestados pelo usuário
     - Exibe contagem de empréstimos

Benefício: Controle completo de empréstimos de usuários

💡 DICA #1: Persistência de Dados
───────────────────────────────────
O sistema atual perde dados quando encerrado. Sugestões:
• Use serialização Java (ObjectInputStream/ObjectOutputStream)
• Use JSON (Gson, Jackson)
• Use banco de dados (SQLite, MySQL com JDBC)
• Use arquivos XML

Exemplo com Serialização:
  // Salvar
  ObjectOutputStream oos = new ObjectOutputStream(
    new FileOutputStream("biblioteca.dat"));
  oos.writeObject(biblioteca);
  oos.close();

  // Carregar
  ObjectInputStream ois = new ObjectInputStream(
    new FileInputStream("biblioteca.dat"));
  biblioteca = (Biblioteca) ois.readObject();
  ois.close();
