                    VALIDAÇÕES IMPLEMENTADAS
                    ═════════════════════════

1. REGISTRAR LIVRO POR ISBN
   • ISBN não vazio ✅
   • ISBN não duplicado ✅
   • Conexão com API com timeout ✅
   • Confirmação de dados ✅

2. REGISTRAR LIVRO MANUALMENTE
   • Título não vazio ✅
   • Autor não vazio ✅
   • ISBN não vazio ✅
   • ISBN não duplicado ✅

3. REGISTRAR USUÁRIO
   • Nome não vazio ✅
   • Email não vazio ✅
   • Email não duplicado ✅

4. EMPRESTAR LIVRO
   • Usuário existe ✅
   • Livro existe ✅
   • Livro está disponível ✅

5. DEVOLVER LIVRO
   • Usuário existe ✅
   • Usuário tem empréstimos ✅
   • Livro está emprestado a este usuário ✅
